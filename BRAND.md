# Sunpebble 品牌手册（v1，2026-07-02）

## 名字

**Sunpebble** = 孙（Sun，孙淑媛·设计与产品）+ 石（pebble，石坤·工程）。
意象：阳光下一颗颗打磨光滑的小卵石——每颗是一个小而美的 app。

## 一句话

> Small, polished apps.

## 分工

- **Kun（石坤）** — engineering
- **Shuyuan（孙淑媛）** — design & product

## 命名约定

- GitHub org：`sunpebble`；repo 名 = app 名小写（如 `dayroll`）
- Bundle id：`com.sunpebble.<app>`；App Group：`group.com.sunpebble.<app>`；内购：`com.sunpebble.<app>.lifetime`
- App 名：一个词，易读易拼（Dayroll ✓）
- 域名：`sunpebblelabs.com`（Cloudflare 托管，官网 = GitHub Pages 自定义域名）
- 支持邮箱：`support@sunpebblelabs.com`（Cloudflare Email Routing 转发到 Gmail）

## 视觉

- Logo：日出 + 卵石（`assets/logo.svg`），太阳在卵石后升起
- 色板：奶油底 `#FFF6E8` / 日光黄 `#F7B733` / 卵石灰 `#6E6E73` / 墨色 `#232733`
- 字体：正文用系统无衬线栈（SF Pro / system-ui），标题用系统衬线栈（New York / Georgia）；不引入 webfont
- App 图标：同一形状语言（圆润、几何、暖色点缀），摆在一起像一窝卵石

## 语气

克制、温暖、不吹嘘。少用感叹号。核心承诺永远显式出现：**private by design**——无账号、无广告、无追踪。

## 明确不做（现阶段）

不注册公司主体、不做 monorepo、不建 org 级 CI 模板、不画完整 VI 手册。
